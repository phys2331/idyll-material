How to Complete this Activity


  1. Click the tabs to go to a specific homework problem page.
    • Note: Not all of the homework problem pages are populated in every homework.
    • The problems are included in Homework 1 through WileyPLUS.
    • Work through the problems by hand first using the given information, key equations, and references listed at the bottom of each page. Then, try to use Python to solve the same problem.
  2. Read the problem and try to answer the questions that are given. These are extra and only there to help!
  3. Use the simulations when provided. These may help with your conceptual understanding of the problem.
  4. Complete the Computation section by following the instructions to the left of the Trinket. This should give you the answers you need to complete the homework!
  5. After completing all of the resources provided, please complete the Feedback Form.
